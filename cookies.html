Cookie Policy

This page explains how Reproduc-io uses cookies and similar technologies on this website.

At the time of publication, this website is designed to operate without non-essential analytics, advertising, or marketing cookies set by Reproduc-io.

The website is a static site hosted on GitHub Pages. Basic technical requests may still be logged by hosting infrastructure for security and delivery purposes.

2. Strictly necessary technologies

If strictly necessary technologies are used to deliver the website, secure the service, or preserve core functionality, they may be processed without a separate opt-in where permitted by applicable law.

3. Optional tools in the future

If Reproduc-io later adds analytics, embedded third-party services, advertising tools, or other non-essential technologies, this policy should be updated before those tools are activated.

Where required by applicable law, a consent mechanism should be implemented before any non-essential cookies or similar technologies are set.

4. Browser controls

Most browsers let you view, block, or delete cookies. Please note that disabling necessary technologies may affect website functionality.
